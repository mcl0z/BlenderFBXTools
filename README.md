# AMI 骨骼动画管理工具（Blender 插件）

这是一个面向 Blender 的动画工作流插件，重点解决以下问题：

- 动画资产沉淀和复用（动作库）
- 不同角色之间的动画迁移（骨骼筛选 + 骨骼映射）
- 多段动作拼接与循环复制（动画合并）
- 常见动画效果快速生成（路径、呼吸、摄像机）

插件入口位置：

- `View3D > Sidebar > 动画工具`

## 功能概览

### 1. 动作库

- 导入 FBX 动作到本地动作库
- 导入/导出 JSON 动作
- 将当前骨架上的动作保存到动作库
- 动作重命名、删除、分类管理

适用场景：

- 建立团队共享的动作素材库
- 在项目中复用历史动画片段

### 2. 动画合并

- 以动作库中“当前选中动作”为来源执行合并
- 支持目标起始帧设置、循环次数设置
- 支持合并模式（覆盖 / 混合）
- 可选保持骨架位置与旋转

适用场景：

- 组合待机、行走、转身等片段形成完整序列
- 快速拉长循环动作时长

### 3. 骨骼映射与骨骼过滤

- 执行合并前可选择要应用动画的骨骼
- 支持源骨骼到目标骨骼的映射
- 支持自动映射和手动搜索映射

适用场景：

- 解决不同命名规范骨架之间的动画迁移
- 只复制局部动画（例如只复制上半身）

### 4. 路径动画

- 创建贝塞尔路径
- 设置目标对象并应用路径跟随动画
- 支持控制点数量和关键参数调整

### 5. 呼吸动画

- 对目标骨骼快速生成呼吸节奏动画
- 可调节幅度、周期、衰减等参数

### 6. 摄像机动画

- 相机抖动动画
- 相机跟随/追踪动画

## 典型使用流程

1. 在“动作库”面板导入 FBX 或 JSON 动作。
2. 在列表中选中要使用的动作。
3. 切到“动画合并”面板设置目标起始帧和循环次数。
4. 按需开启骨骼映射并调整映射关系。
5. 点击“应用动作库选中动画”完成合并。

## 版本与安装

- 插件目标 Blender 版本：`3.6+`
- 推荐安装方式：在 Blender 偏好设置中安装 `dist/` 目录下打包好的 ZIP

本仓库打包命令：

```bash
python3 build_addon_zip.py
```
